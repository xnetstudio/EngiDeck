# EngiDeck

**一站式运维管理平台** — 集多协议连接、AI 助手、安全凭证于一体的现代化运维工具。

## 核心功能

### 多协议支持

一个工具连接所有服务：

| 协议               | 功能                            |
| ------------------ | ------------------------------- |
| SSH                | 完整 PTY 支持，媲美原生终端体验 |
| SFTP / FTP         | 可视化文件管理，拖拽上传下载    |
| MySQL / PostgreSQL | 数据库浏览、SQL 查询执行        |
| Redis              | 键值浏览、数据管理              |
| Docker             | 容器启停、日志查看、终端连接    |
| VNC / RDP          | 远程桌面连接                    |
| Serial             | 串口设备调试                    |
| Nginx              | 配置文件可视化管理              |

### AI 智能助手

内置 AI 助手，让运维更智能：

- **Chat 模式**：与 AI 对话获取建议和解答
- **Agent 模式**：AI 自动执行命令完成任务
- **风险检测**：智能分析命令风险等级，敏感操作需确认
- **支持 DeepSeek 和阿里云百炼**

<img width="1052" height="737" alt="截屏2026-01-22 22 15 56" src="https://github.com/user-attachments/assets/f237342a-2d8c-4b2a-ad9c-b9a60d59289d" />



### 安全密码管理

您的凭证安全是我们的第一优先级：

- **AES-256 加密**：军用级加密标准保护所有凭证
- **本地存储**：数据永远只存储在您的设备上
- **主密码保护**：Argon2id 密钥派生算法
- **SSH 密钥管理**：一键生成并部署公钥到服务器
- **主机密钥验证**：防止中间人攻击

### 高效工作流

提升您的工作效率：

- **多组织管理**：创建多个组织空间，独立管理不同项目或客户
- **标签分类**：灵活的标签系统，快速定位目标资源
- **独立弹窗**：将终端弹出为独立窗口
- **分屏操作**：多任务并行处理
- **颜色标记**：为标签页设置不同颜色，区分生产/测试环境
- **水印标记**：自定义水印，避免误操作
